## Código Q 

El Código Q es un conjunto de abreviaturas de tres letras que se utiliza en las comunicaciones por radio para transmitir información compleja de manera rápida y precisa.

Originalmente fue creado por el gobierno británico en 1909 para facilitar la comunicación entre barcos y estaciones costeras que hablaban diferentes idiomas, pero hoy es un estándar mundial en la radioafición, la navegación marítima, la aviación y, por supuesto, es una herramienta muy útil para usuarios de Meshtastic en situaciones de emergencia.

<div style={{ marginLeft: 'auto', marginRight: 'auto', width: 'fit-content' }}>
| Código | Significado | Ejemplo de uso |
|:---:|:---:|:---:|
| QSL | Mensaje recibido y entendido | "QSL, avanzando a zona 2" |
| QTH | Ubicación o posición actual | "¿Cuál es tu QTH?" / "QTH: Base Sur" |
| QRV | Estoy listo / atento / operativo | "Unidad 1 QRV en el canal" |
| QRT | Fin de la comunicación / Apago nodo | "Misión cumplida, QRT" |
| QSY | Cambiar de canal o frecuencia | "QSY al canal de Emergencia" |
| QRX | Espere un momento (Stand-by) | "QRX 5 min, recargando batería" |
| QSO | Contacto o charla establecida | "Iniciando QSO con base central" |
| QRZ | ¿Quién me llama? (No identifiqué ID) | "QRZ? Repita mensaje" |
</div>

**REGLAS DE ORO EN EMERGENCIAS**
1. Piense, luego escriba: Sea breve y conciso.
2. Confirme: Si recibe una instrucción crítica, responda siempre con un QSL.
3. Prioridad: Deje el canal libre si escucha un mensaje de emergencia (SOS).
4. Estado de Batería: Reporte si su nodo entra en batería baja antes de quedar QRT.
